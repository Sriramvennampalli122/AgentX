# **App Name**: AgentX Terminal

## Core Features:

- Autonomous Reasoning Engine: An LLM-driven core that uses tool-calling to orchestrate complex multi-step tasks independently.
- Authenticated Persistence: Store all agent sessions, message history, and tool execution logs securely in a Firestore database, accessible via Google Sign-In.
- Live Search Integration: A specialized tool that allows the agent to scrape the web via Cloud Functions to retrieve up-to-the-minute information.
- Sandboxed Execution Tool: An environment to run and test JavaScript snippets generated by the agent for data processing and algorithmic verification.
- Streaming Progress Monitor: A real-time UI dashboard showing the agent's 'thinking' process, tool triggers, and final answers in a tabbed terminal layout.
- Integrated Math & Logic Tool: Dedicated calculation tool for the agent to perform precision arithmetic and logic evaluations without LLM hallucination.
- Firestore History Storage: Detailed archiving of all agent steps and user tasks in Firestore for easy session retrieval.

## Style Guidelines:

- Primary color: Electric Indigo (#6366F1), evoking a high-tech intelligence feel.
- Background: A deep, midnight obsidian (#0F0F12), heavily desaturated and dark for a command-center aesthetic.
- Accent color: Bright Vivid Blue (#0EA5E9), used for highlights, tool status badges, and interactive terminal elements.
- Font pairing: 'Space Grotesk' for modern, tech-focused headings and 'Inter' for highly legible body text.
- Code display: 'Source Code Pro' used exclusively for agent 'thinking' logs and sandbox output areas.
- A split-pane grid layout featuring a collapsible history sidebar and a tabbed execution panel for multi-channel tool visibility.
- Low-latency slide-in animations for thinking steps and subtle pulsing dot indicators for 'agent working' states.
